# Удаление тестовых заданий и примеров

## Как удалить тестовые задания

Тестовые задания можно удалить через API endpoint:

### 1. Удаление тестовых заданий (is_test=True)

Отправьте DELETE запрос на:
```
DELETE /api/admin/delete-test-tasks
```

Или через curl:
```bash
curl -X DELETE https://ваш-домен/api/admin/delete-test-tasks
```

### 2. Удаление примеров заданий (для страницы "Создать")

Отправьте DELETE запрос на:
```
DELETE /api/admin/delete-example-tasks
```

Или через curl:
```bash
curl -X DELETE https://ваш-домен/api/admin/delete-example-tasks
```

## Что изменилось

1. ✅ Комиссия приложения изменена с 5% на 10%
2. ✅ Комиссия теперь вычитается с исполнителя задания, а не с создателя
3. ✅ Реферальная система: 5% с каждого выполненного задания отдаётся рефералу
4. ✅ Правила пользования обновлены с информацией о комиссии 10% и реферальной системе 5%

## Важно

- Комиссия 10% вычитается автоматически при зачислении награды исполнителю
- Реферальная комиссия 5% начисляется автоматически, если пользователь пришёл по реферальной ссылке
- Все изменения применены и готовы к использованию
